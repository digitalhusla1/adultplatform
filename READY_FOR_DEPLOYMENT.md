## üö® IF SOMETHING GOES WRONG

### Site not loading
1. Clear browser cache (Ctrl+Shift+R)
2. Check Netlify dashboard for deploy errors
3. Check deploy logs for details

### Forms not working
1. Verify `<form netlify>` attribute
2. Check form `name` attribute
3. Redeploy site
4. Clear browser cache

### Styling looks broken
1. Hard refresh browser (Ctrl+Shift+R)
2. Clear browser cache
3. Wait 5 minutes (cache clearing)
4. Refresh again

### Age modal not showing
1. Clear localStorage in DevTools
2. Refresh page
3. Check console for errors (F12)

See **EDITING_GUIDE.md** for more troubleshooting!
